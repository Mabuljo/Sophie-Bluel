Projet Sophie Bluel Architecte d'intérieur

Ce projet a pour but de créer une page web dynamique pour le site internet d'une architecte d'intérieur en utilisant JavaScript et en communiquant avec une API.

Apprendre à travailler sur la partie Front-End du site, en développant la page de présentation des travaux de l'architecte à partir du HTML fourni. Cela impliquera la gestion des événements utilisateurs et la manipulation des éléments du DOM avec JavaScript.

Apprendre à communiquer avec une API, ce qui est crucial pour les applications web modernes.
Apprendre à créér une page de connexion pour l'administrateur du site ainsi qu'une modale permettant d'uploader de nouveaux médias. 

Ces éléments seront développés from scratch, offrant l'opportunité de mettre en pratique les compétences en JavaScript de manière approfondie et
avoir accès à un code back-end pour tester les fonctionnalités.
